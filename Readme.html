RTor - A JQueryUI Demo

This is a demo based on some ancient Tektronics terminal hacks from many(!) years ago. I revived them a while ago in Flash, and when Flash was starting to get passe, tried them in HTML.

The actual demo might keep you amused for thirty seconds or so...but the code does show how to use various JQueryUI elements, and serves as worked example of an HTML5 application. I'm also using it as a way to become familiar with Github.
